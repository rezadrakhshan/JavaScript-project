Number guessing game


Welcome

The number guessing game is a simple and entertaining game where the objective is for the player to guess a randomly chosen number within a specified range. The game provides feedback on each guess, indicating whether the guess is too high, too low, or correct. This game can be played with a computer or among players, and it is a great exercise for practicing logic and problem-solving skills.

Rules of the Number Guessing Game

  1. Range Selection:
    • The game will set a range of numbers (e.g., 1 to 100).
    • The computer will randomly choose a number within this range.
  2. Number of Attempts:
    • Players will have a limited number of attempts to guess the correct number.
    • The number of allowed attempts can vary based on the difficulty level. For example, in a range of 1 to 100, a common limit is 10 attempts.
  3. Making a Guess:
    • The player makes a guess within the specified range.
    • The computer provides feedback for each guess:
      • Too High: If the guess is higher than the target number.
      • Too Low: If the guess is lower than the target number.
      • Correct: If the guess matches the target number.
  4. Winning the Game:
    • The player wins the game if they guess the correct number within the allowed number of attempts.
  5. Losing the Game:
    • The player loses the game if they fail to guess the correct number within the allowed number of attempts.

Strategies

  1. Binary Search:
    • Start with the middle number of the range.
    • Adjust the range based on whether the guess is too high or too low.
    • This strategy helps to narrow down the possible numbers quickly.
  2. Track Previous Guesses:
    • Keep a record of previous guesses to avoid repeating them and to refine the range of possible numbers.
